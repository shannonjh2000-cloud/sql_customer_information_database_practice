# SQL Projects Portfolio
A collection of SQL projects analyzing and managing data. These projects highlight a wide range of SQL skills, from basic queries and aggregations to advanced data operations. Which showcase my knowledge in both data analysis and database management.
SQL Skills & Concepts Demonstrated
Data Retrieval & Filtering

SELECT, DISTINCT – Retrieve specific columns and unique records from a table

WHERE – Filter rows based on specific conditions

LIMIT – Restrict the number of results returned

CASE – Apply conditional logic within queries

Aggregations & Analysis

COUNT(), SUM(), MAX(), MIN(), AVG(), ROUND() – Perform calculations and summarize data

GROUP BY, ORDER BY, HAVING – Aggregate, sort, and filter grouped results

Database Management

CREATE TABLE – Define and create new tables

INSERT INTO – Add new records to tables

UPDATE – Modify existing records

ALTER TABLE, ADD COLUMN – Change table structure and add new columns

DELETE – Remove records from a table

Combining Data

JOIN – Merge data from multiple tables based on relationships

UNION – Combine results from multiple queries into a single result set
